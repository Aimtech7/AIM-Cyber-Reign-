# AIM: Cyber Reign

> Cyberpunk 3D game — Python + Ursina

**Author:** Aimtech | **Version:** 0.5.0 | **Phase 5 — Mission & Objective System**

---

## 🎮 Features

- First-person movement (WASD, sprint, jump)
- Cyberpunk 3D world (buildings, pillars, walls, platforms, terminals)
- Hacking mini-game with terminal states
- Security drones with 3-state AI + alert system
- Player health with regen
- **Mission system** — "SECTOR BREACH": breach 2 targets + reach extraction
- **Win/lose conditions** — mission complete / failed with restart
- **Extraction zone** — glowing teal pad, only activates after objectives
- **Mission HUD** — title, objective, progress, feedback messages
- Scene system, Docker support

## 🚀 Run

```bash
pip install -r requirements.txt
python main.py
```

| Key | Action |
|---|---|
| W/A/S/D | Move |
| Mouse | Look |
| Space | Jump |
| Shift | Sprint |
| E | Interact/Hack/Extract |
| ESC | Menu/abort hack |
| R | Restart (on end screen) |
